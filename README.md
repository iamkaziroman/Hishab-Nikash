# Hishab-Nikash
Track your income, manage daily expenses, and boost your savings effortlessly!

## Hishab-Nikash Website
A modern static income tracker web app with:

- Login page with protected access
- Transaction manager for income and expenses
- Editable recurring transaction support
- Live analytics, trend forecast, and category insights
- Advanced settings for theme, accent, currency, animations, and goals
- CSV export and local persistence via `localStorage`

## Files
- `login.html` — secure login screen
- `app.html` — dashboard and tracker interface
- `styles.css` — animated UI styling and responsive layout
- `script.js` — all app interactions, state persistence, and settings
